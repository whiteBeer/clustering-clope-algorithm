## Алгоритм кластеризации данных CLOPE

Программа запускается в среде node.js: https://nodejs.org/en/download

Чтобы запустить программу, выполним команду: 
```
 node app.js 
```

По умолчанию исходные данные находятся в файле
[agaricus-lepiota.data](agaricus-lepiota.data)

После выполнения программы, результат записывается в файл 
[agaricus-lepiota.data.cluster](agaricus-lepiota.data.cluster)

**или с параметрами:** 

```
 node app.js <fileName> <fileLineLength> <r>
```

где,

* `<fileName>` - название входного файла
* `<fileLineLength>` - длина строки (транзакции) во входном файле (с учетом сивола переноса строки `\n`)
* `<r>` - коэффициент отталкивания в алгоритме CLOPE

например:
```
 node app.js ./agaricus-lepiota-another.data 46 2.6
```

----
Следует отметить что для корректной работы программы
во входном файле все признаки должны быть
разделены символом ",". Длина строк должна быть одинаковой, 
и в конце строки находиться символ `\n`

Пример входного файла:

```
e,x,s,y,t,e
p,y,r,y,f,e
e,z,s,y,h,j
e,z,s,y,h,j
e,z,s,y,f,j
p,y,r,y,f,e
```
